## What is Cats Blender Plugin?

The Cats Blender Plugin is a powerful add-on for Blender designed to simplify and speed up the process of importing, fixing, and optimizing 3D models for use in VRChat. It supports a wide range of model formats (such as MMD, XNALara, Mixamo, Source Engine, Unreal Engine, DAZ/Poser, Blender Rigify, Sims 2, Motion Builder, and 3DS Max) and reduces complicated manual steps involved in preparing avatars.

## Main Features
- One-click model optimization
- Automated creation of lip sync (viseme) shapes
- Eye tracking setup
- Automated mesh decimation (with shape key support)
- Custom model creation workflows
- Texture atlas generation
- Root bone creation for Dynamic Bones
- Material optimization
- Auto translation of shape keys, bones, materials, and meshes
- Bone merging to reduce bone counts
- Built-in auto updater

## User Experience
The goal of the plugin is to save creators countless hours by providing easy one-click or automated solutions to problems that previously required lots of manual tweaks. For example, transforming "broken" models into VRChat-ready avatars now takes minutes instead of hours.

## Installation
1. Download the latest release from the official repository.
2. **Do not extract the zip!** Install it directly in Blender via the add-ons panel.
3. Locate the new CATS tab in Blender (press 'N' if needed)

For help and community support, join their [Discord server](https://discord.gg/f8yZGnv).

## Example Advanced Features
- Fixes models by reparenting/removing bones, renaming/translating objects, combining meshes, optimizing for full-body tracking, and more.
- Pose mode for testing bones and saving shape keys
- Bone and mesh translation (Japanese->English), applying transformations, deleting constraints, and more
- Smart decimation process that simplifies models for faster VRChat upload
