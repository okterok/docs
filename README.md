# docs

Welcome to the **docs** repository!

This repository contains documentation, guides, and reference materials. All `.md` files can be browsed right here on GitHub, and are also accessible as static web pages via [GitHub Pages](https://okterok.github.io/docs/) if enabled.

## About This Project

This repository exists mainly as a personal sandbox for experimenting with tools like [Obsidian](https://obsidian.md/) for Markdown note-taking and using [GitHub Pages](https://pages.github.com/) for publishing Markdown files as a static site. It's intended for learning, tinkering, and exploring workflows.

## How to Access

- **Browse Markdown files:** View any `.md` file directly in this repository.
- **GitHub Pages:** Access the documentation site at [okterok.github.io/docs](https://okterok.github.io/docs/) (if enabled).

This is an open documentation resource. While it is open source, external contributions are not expected, but issues or suggestions are welcome.
